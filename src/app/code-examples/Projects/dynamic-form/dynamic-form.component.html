Dynamic Form

This is an example of the dynamic form that I created for the All In CMS. It's purpose is so that you can add, remove, and rearange sections that would later be displayed on the mobile apps. This way you can add text, an image, an input box, etc and fully design the pages and content the app would deliver straight from the CMS. This is a watered down version however, as the full version had many more different sections you could add, such as questions and answers, video, and much more, as well as background options, next page destinations, etc.

A proper write-up on how it works and why I coded it in the way I did will be coming soon, but for now you can find the code at my GitHub page for this website.
