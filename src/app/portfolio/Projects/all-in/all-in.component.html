All In Software

For this project I was tasked with creating an HR training application for mobile phones, accompanied by an online content management system. This software is used by some major companies, such as GKN, a major multinational automotive and aerospace components company. The Digital Brand Accelerator website advertising this software can be seen here.

Created from the ground up, I decided to go with natively coded phone apps so that they feel natural to the platform, as well as Node.js, Angular 5, and Firebase for the CMS.

The CMS

Users

Users can be added by admins via a form page, and existing users can be edited and added to any number of groups that will change what modules are displayed to them via the app.

Groups

Groups are used to separate and refine what content the users will be able to access on the app. These groups are assigned to modules.

Modules

Modules are nested with sub-modules and lessons, with lessons being the main content the users see. Lessons consist of text, images and/or questions that can be rearranged on the fly using the custom built dynamic form.

The App

Login

Only users manually approved and added by management are allowed to access the content of the app, so a login screen is required. Fully integrated with a lost password reminder as well.

Modules

Upon signing in the users will be presented with a list of Modules that are approved for them via the group they are assigned. Modules navigate down to sub-modules, and then to lessons.

Dynamic Content

The modules list, as well as all lessons are all dynamically created based on the content delivered by the CMS. Lessons will navigate to the next lesson once completed based on what was specified. Data on these completed lessons is then sent back to the CMS so that managers can collect data on their users.
